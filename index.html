About

사용자 경험을 코드로 구체화합니다.

HTML의 시맨틱 구조, CSS의 반응형 레이아웃, JavaScript의 DOM 조작을 중심으로 웹의 기본기를 학습하고 있습니다. 작은 기능이라도 이벤트, 상태, 렌더링의 흐름이 명확하게 드러나도록 구현하는 것을 중요하게 생각합니다.

이 페이지는 외부 UI 라이브러리 없이 제작되었으며, GitHub 저장소 데이터를 비동기로 가져와 프로젝트 카드로 보여줍니다.
